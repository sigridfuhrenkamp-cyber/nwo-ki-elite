## 📝 Dokumentations-Standard

### Für jeden Post:
- **URL:** Direkter Instagram-Link
- **Datum/Zeit:** Original-Postdatum + relative Zeitangabe (z.B. "91 Wochen alt")
- **Likes:** Anzahl der Likes
- **Caption:** Vollständiger Originaltext
- **@-Mentions:** Alle erwähnten Accounts
- **Hashtags:** Vollständige Hashtag-Liste
- **Kommentare:** Öffentlich sichtbare Kommentare (Stichprobe)
- **Screenshots:** SHA-256 gehasht
- **Metadaten:** Erhebungszeit, Gerät, Browser, Navigationspfad

### Namensschema für Artefakte:
```
YYYYMMDD_HHMMSS_TZ__ig__wirliebentanzen1love1family__<action>__<seq>.<ext>
```
